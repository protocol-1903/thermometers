# This mod alters existing pipes. You will need another mod to add new pipes.

Allows you to read the temperature of fluids inside any tanks. The signal produced is configurable. 
Please report any bugs here or on [github](https://github.com/protocol-1903/thermometers)



If you have a mod idea, let me know and I can look into it.
